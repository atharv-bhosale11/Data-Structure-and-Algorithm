Data Structures in C
Overview

This repository contains implementations of fundamental Data Structures using the C programming language. The goal of this repository is to understand the internal working of various data structures, improve problem-solving skills, and strengthen programming fundamentals.

Topics Covered
Linked List
Singly Linked List
Doubly Linked List
Circular Linked List
Operations:
Insert First
Insert Last
Insert At Position
Delete First
Delete Last
Delete At Position
Display
Count
