# Singly Linked List in C++

This repository contains the implementation of a **Singly Linked List** using C++. The project demonstrates the fundamental operations of a linked list and provides a strong foundation for understanding dynamic data structures.

## Features

* Insert Node at First Position
* Insert Node at Last Position
* Insert Node at Specific Position
* Delete First Node
* Delete Last Node
* Delete Node from Specific Position
* Display Linked List Elements
* Count Total Number of Nodes

## Concepts Covered

* Classes and Objects
* Dynamic Memory Allocation
* Self-Referential Structures
* Pointers and Pointer Manipulation
* Linked List Traversal
* Memory Management
* Data Structure Implementation

## Operations Implemented

### Insertion Operations

* InsertFirst()
* InsertLast()
* InsertAtPos()

### Deletion Operations

* DeleteFirst()
* DeleteLast()
* DeleteAtPos()

### Utility Operations

* Display()
* Count()

## Sample Output

```text id="n4b51w"
|11| -> |21| -> |51| -> |101| -> NULL
Number of Nodes : 4

|11| -> |21| -> |51| -> |101| -> |111| -> NULL
Number of Nodes : 5
```

## Learning Objectives

This project helps in understanding:

* How nodes are created dynamically.
* How pointers connect nodes together.
* How insertion and deletion operations work internally.
* How linked lists differ from arrays.
* How memory is managed dynamically in C++.

## Requirements

* C++ Compiler (GCC/G++, Clang, MSVC)
* Any C++ IDE or Code Editor
