# Data Structures in C++

## Overview

This repository contains implementations of fundamental Data Structures using the C++ programming language. The purpose of this repository is to build a strong understanding of data organization, memory management, object-oriented programming, and efficient problem-solving techniques. Each data structure is implemented from scratch to provide a clear understanding of its internal working and operations.

## Topics Covered

### 1. Linked List

* Singly Linked List
* Doubly Linked List
* Circular Singly Linked List
* Circular Doubly Linked List
* Insertion Operations
* Deletion Operations
* Traversal
* Searching
* Counting Nodes

### 2. Stack

* Stack Using Array
* Stack Using Linked List
* Push Operation
* Pop Operation
* Peek Operation

### 3. Queue

* Queue Using Array
* Queue Using Linked List
* Circular Queue
* Enqueue Operation
* Dequeue Operation

### 4. Tree

* Binary Search Tree (BST)
* Inorder Traversal
* Preorder Traversal
* Postorder Traversal
* Search Operation

### 5. Searching Algorithms

* Linear Search
* Binary Search

### 6. Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### 7. Hashing

* Hash Table Implementation
* Collision Handling Techniques
