# Singly Linked List in Java

This repository contains the implementation of a **Singly Linked List Data Structure in Java**. The project is designed to demonstrate the core concepts of linked lists, including node creation, insertion, deletion, traversal, and position-based operations using Object-Oriented Programming principles.

## 📚 Topics Covered

* Node Structure
* Dynamic Memory Allocation using Objects
* Insert First
* Insert Last
* Insert At Position
* Delete First
* Delete Last
* Delete At Position
* Display Linked List
* Count Total Nodes
* Menu Driven Implementation
* Object-Oriented Design

## 🚀 Features

* Clean and beginner-friendly code
* Fully object-oriented implementation
* Position-based insertion and deletion
* Dynamic node management
* Interactive menu-driven application
* Suitable for Data Structures and Algorithms practice

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
