# Data Structures in Java

## Overview

This repository contains the implementation of various Data Structures using Java. It focuses on understanding data organization, object-oriented programming concepts, and efficient problem-solving techniques.

## Topics Covered

1. Linked List

   * Singly Linked List
   * Doubly Linked List
   * Circular Singly Linked List
   * Circular Doubly Linked List

2. Stack

   * Stack Implementation

3. Queue

   * Queue Implementation
   * Circular Queue

4. Trees

   * Binary Search Tree (BST)
   * Tree Traversals

5. Searching Algorithms

   * Linear Search
   * Binary Search

6. Sorting Algorithms

   * Bubble Sort
   * Selection Sort
   * Insertion Sort
   * Merge Sort
   * Quick Sort

## Technologies Used

* Java
* JDK
* Visual Studio Code / IntelliJ IDEA
* Git & GitHub

## Learning Objectives

* Understand Data Structures using Java
* Strengthen Object-Oriented Programming concepts
* Improve coding and problem-solving skills
* Prepare for coding interviews and placements
