# Java Data Structures

A comprehensive collection of Data Structure programs implemented in Java. This repository is designed for students, beginners, and developers who want to learn, practice, and master Data Structures using object-oriented programming concepts.

## Topics Covered

### Linked Lists

* Singly Linear Linked List
* Singly Circular Linked List
* Doubly Linear Linked List
* Doubly Circular Linked List
* Insertion Operations
* Deletion Operations
* Search Operations
* Traversal Operations

### Stacks

* Stack using Arrays
* Stack using Linked List
* Push Operation
* Pop Operation
* Peek Operation

### Queues

* Linear Queue
* Circular Queue
* Queue using Linked List
* Enqueue Operation
* Dequeue Operation

### Trees

* Binary Tree
* Binary Search Tree (BST)
* Tree Traversal Techniques

  * Inorder
  * Preorder
  * Postorder

### Searching Techniques

* Linear Search
* Binary Search

### Sorting Techniques

* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort

### Recursion

* Basic Recursive Programs
* Mathematical Recursion
* Recursive Problem Solving

### Hashing

* Hash Table Implementation
* Collision Handling Concepts

## Features
